# sdihub-php-client
PHP API client for SdI Shellrent Hub

## Installazione
Aggiungere nel composer.json

`"repositories": [
    {
        "type": "vcs",
        "url": "git@github.com:ShellrentSrl/sdihub-php-client.git"
    }
],`
  
`"require": {
  "ShellrentSrl/sdihub-php-client": "dev-master"
}`
